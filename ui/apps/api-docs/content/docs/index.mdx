The Inngest REST API lets you interact with events, function runs, environments, and more programmatically.

## Versions

| Version | Status | Base URL |
| --- | --- | --- |
| [v1](/v1) | Stable | `https://api.inngest.com/v1` |
| [v2](/v2) | Stable (In-development) | `https://api.inngest.com/v2` |

## Authentication

All API requests require authentication. See the [Authentication](/authentication) guide for details.

| Environment | URL |
| --- | --- |
| Production | `https://api.inngest.com` |
