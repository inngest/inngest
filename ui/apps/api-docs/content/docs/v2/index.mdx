---
title: v2 REST API
---

Our v2 API is in active development and supports both the Developer API and the Partner API. Both sets of API endpoints are included under the `v2` prefix.

## Developer API

The developer API is used for all key account-level actions, including runs, events, insights, and resource management like webhooks.

## Partner API

The partner API is used for products that wish to programmatically create Inngest accounts on behalf of their own users, providing a self-contained, and transferable account for their own customers.
